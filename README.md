# Flutter + Node.js POS Application

This project is a **Flutter-based POS (Point of Sale) application** with a **Node.js (Express) REST API backend**.  
The Flutter app consumes APIs from the Node backend to manage products, cart, and billing.

---

## Tech Stack

### Frontend
- Flutter (Dart)
- Provider (State Management)
- HTTP package

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
